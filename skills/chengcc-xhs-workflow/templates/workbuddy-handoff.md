# WorkBuddy Handoff

## Goal

Create a platform draft. Do not final-publish unless the user explicitly asks in the current turn.

## Target Platform

- Platform code: xiaohongshu / douyin / wechat_channels / wechat_official_account
- Draft surface:
- Logged-in creator account:
- Review result: pass / revise / block

If review is `revise` or `block`, do not create the draft until the creator approves the fix.

## Platform Operation

Use the matching draft route:

- `xiaohongshu`: open Xiaohongshu Creator Center image-text publishing page.
- `douyin`: open Douyin Creator Service / publishing page for image-text or video.
- `wechat_channels`: open WeChat Channels Assistant / publishing page.
- `wechat_official_account`: open WeChat Official Account editor.

Then:

- Upload or place images in the order below.
- Fill title, body/caption/article text, tags/topics, and any required topic fields exactly.
- Mark AI-generated / AI-assisted content according to platform options when final cards were generated by Image 2 or other AI tools.
- If no AI label field is visible but the review requires disclosure, add this light disclosure to the body/caption: `图片由 AI 辅助生成，内容来自我的真实经历整理。`
- Add source/fiction/staged/commercial/activity labels when the review says they are needed.
- Save as draft or stop on the pre-publish review page.
