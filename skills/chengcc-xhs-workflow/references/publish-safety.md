# Publish Safety

## Default Rule

The creator controls final publishing manually.

The workflow should provide a checklist, not operate the platform by default.

## 2026 Review Gate

Run this after image generation / final prompt output and before the manual publish checklist.

First select the target platform:

- `xiaohongshu` for 小红书
- `douyin` for 抖音
- `wechat_channels` for 微信视频号
- `wechat_official_account` for 微信公众号

If the user cross-posts, run the gate once per platform. Use `docs/platform-publish-rules-2026.md` for platform-specific checks.

### 1. AI Labeling

- If the images are primarily generated by Image 2 or other AI tools, mark or disclose AI-generated / AI-assisted content where the platform provides the option.
- Do not remove, hide, or falsify AI generation labels or metadata.
- If the platform does not expose a label option, include a light disclosure in the body only when needed: `图片由 AI 辅助生成，内容来自我的真实经历整理。`

### 2. Authenticity

- No fake personal experience.
- No fake AI chat screenshot.
- No fake comments, likes, endorsements, or outcomes.
- No AI托管/auto-operated account behavior.
- No pretending an AI-generated person/object is real.

### 3. Originality And Rights

- Do not copy competitor text, titles, card layout, or images.
- Do not use borrowed IP, celebrity likeness, brand logos, copyrighted characters, or unlicensed artwork.
- Use only original orange IP for 澄Cc.
- If screenshots are used, remove private names, avatars, IDs, company/client info, and chat details.

### 4. Content Claims

Reject or revise if the post contains:

- medical or mental-health diagnosis/treatment promises
- legal, tax, financial, investment, or career guarantees
- absolute outcome claims such as `一定有效`, `彻底治好`, `保证上岸`
- misleading before/after claims
- moral judgment or harassment toward a real person

### 5. Platform Publishing Hygiene

- Target platform is selected before the manual publish checklist.
- Title/cover exists and is readable for the target platform.
- Body/caption exists and adds new value beyond card text.
- Tags/topics are relevant, not spammy.
- No illegal/sensitive content.
- No private information.
- No external traffic bait unless compliant with platform rules.
- No low-quality batch content or repeated template spam.
- Platform-specific rules have been checked from `docs/platform-publish-rules-2026.md`.
